## 🧰 New Tools and Software Updates

### 🚀 Data Release

A collection of over 1,000 samples from the 1K Genome Project Collection was re-sequenced using high-output flow cells on Oxford Nanopore's PromethION instrument. The data release includes raw signal data (FAST5), alignment (CRAM), and variant calls (VCF). If you're interested in using this data for your own research, you can access it for free on the [Registry of Open Data on AWS](https://registry.opendata.aws/1kg-ont-vienna/).

<!--truncate-->

### 🚀 Dorado Update

The latest Dorado release (v1.1.0 – July 31, 2025) delivers a major performance upgrade, with up to 30% faster basecalling using the HAC model across a broad range of Nvidia GPUs, including Blackwell, Ada, and Ampere. The release also includes speed improvements for SUP basecalling models running on Hopper GPUs. This version also adds support for primer detection and removal from 10X Genomics single-cell libraries.

More information: Available on [GitHub](https://github.com/nanoporetech/dorado/releases/tag/v1.1.0)
