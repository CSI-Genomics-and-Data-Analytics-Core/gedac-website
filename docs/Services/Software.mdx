# Software Development

Welcome to our Software Development page. We are happy that you made it here. This page will walk you through the services we offer, the benefits of working with us, and how to get started working together.

## What We Offer

We offer access to highly trained software engineers who have experience working with scientists to translate their ideas into code. Our software engineers have experience building:

- Database management systems
- Interactive analysis tools
- Scientific web applications
- Research lab websites

## Data Pipelines We Develop

We build robust, automated data pipelines to help labs move from raw data to actionable insights faster and more reliably. For examples, see our April 2026 newsletter: [**GeDaC Newsletter - April 2026**](/news/newsletter-apr-2026).

<details>
<summary>Pipeline: Spatial Imaging Pipeline Automation</summary>

We developed an end-to-end pipeline to process spatial imaging datasets in a fully reproducible manner. Researchers upload the images and a short config file; processing runs automatically on dedicated research infrastructure and results are delivered back to the project folder.

</details>

<details>
<summary>Pipeline: Spatial Biology Core (SBC) Automated Analysis</summary>

We modernized a traditionally manual, Excel-based workflow into a structured, high-throughput pipeline where datasets are combined, biological markers are standardized, and cell-level data is rolled up into patient summaries — consistently, every time.

</details>

### Engineering Standards We Bring to Pipelines

- **Containerized Execution**: Using Docker to ensure pipelines run identically across environments.
- **Hybrid Cloud Workflows**: Integrating AWS with on-premise compute resources.
- **Manifest-Driven Runs**: Using structured manifests to trigger automated data synchronization and result delivery.
