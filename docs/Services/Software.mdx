# Software Development

Welcome to our Software Development page. We are happy that you made it here. This page will walk you through the services we offer, the benefits of working with us, and how to get started working together.

## What we Offer

We offer access to highly trained software engineers who have experience working with scientists to translate their ideas into code. Our software engineers have experience building:

- Database management systems
- Interactive analysis tools
- Scientific web applications
- Research lab websites

## Why Choose Us?

The best way to answer this question is to show you what we have built.

<details>

<summary>Project 1: Genome Instability Viewer</summary>

[**Genome Instability Viewer**](https://gi-viewer.pittlabgenomics.com/home) is an interactive web application that integrates genomic instability scores and copy number signatures with molecular and clinical features from patient data.

</details>

<details>

<summary>Project 2: Cloud Flow</summary>

[**CloudFlow**](https://www.cloudflow.gedac.org/) is a web application that allows NUS faculty, staff and students to run best-practice bioinformatic pipelines in the cloud. For more information about this application, please visit our [online documentation](https://docs.cloudflow.gedac.org/)

</details>

<details>

<summary>Project 3: N2CR Avatar Portal</summary>

[**N2CR Avatar Portal**](https://n2cr.gedac.org/login) is a web application that allows users to add, update, and query model records from the N2CR patient-derived-model database. For more information about this application, please visit our [online documentation](https://n2cr.gedac.org/guide).

</details>


<details>

<summary>Project 4: S3 Cost Calculator</summary>

[**S3 Calculator**](https://www.s3.gedac.org/) is a web application written in Python that helps researchers estimate the cost of storing and retrieving genomic data from different Amazon S3 storage classes. 

</details>
<details>
<summary>Project 5: EnsembleFit Portal</summary>

[**EnsembleFit**](https://www.ensemblefit.pittlabgenomics.com/) Implemented the MutSig web-app as part of the EnsembleFit project, enabling users to process, analyze, and download mutation signature insights.

</details>

<details>
<summary>Project 6: GeDaC Connect</summary>

[**GeDaC Connect**](https://connect.gedac.org/) is a platform that allows users to track, manage and analyze data received by our sequencing provider in Singapore.

</details>

<details>

<summary>Project 7: Lab website(s)</summary>

We develop professional websites for academic labs that effectively showcase their research, team members, and publications.  Check out a sample of our work by viewing the lab website of [**Dr. Jason J. Pitt**](https://www.pittlabgenomics.com/) from the Cancer Science Institute of Singapore.

</details>

If you made it this far, it sounds like you might be ready to start a conversation. To get the ball moving, send us an email introducing yourself and a member of our team will respond shortly. We look forward to the possibility of working with you and learning more about your software needs.
